## <i class="bi bi-play-fill"></i> Executing
<P1> 
to run Romine Mini, you will just need to enter this command in your terminal
```shell
python Romine_Mini.pyc
``` 
<br />
or, if you want to do this programmaticaly, do this:
<Code>
    import subprocess

    result = subprocess.run("pythono Romine_Mini.pyc", shell=False, capture_output=False, text=False)
    print(result.stdout)

</Code>
